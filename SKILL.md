# 产品规划与功能设计 Skill

基于 Lenny's Podcast 269期产品管理播客精华，辅助产品规划和功能设计。

## 使用方法

当你需要：
1. 产品规划 - 输入业务目标，输出产品路线图
2. 功能设计 - 输入功能需求，输出详细设计方案
3. 案例参考 - 输入场景，匹配相关播客案例

## 核心框架

### 1. 产品规划框架
```
1. 背景与问题定义
   - 用户痛点
   - 市场规模
   - 竞争格局

2. 解决方案
   - 核心价值主张
   - 关键假设
   - MVP 范围

3. 商业模式
   - 定价策略
   - 获客渠道
   - 成本结构

4. 增长策略
   - 指标体系
   - 增长飞轮
   - 扩展路径
```

### 2. 功能设计框架
```
1. 功能概述
   - 功能名称
   - 解决的问题
   - 核心价值

2. 用户故事
   - 角色
   - 场景
   - 期望

3. 详细设计
   - 信息架构
   - 交互流程
   - 边缘情况

4. 成功指标
   - 核心指标
   - 监控方案
```

### 3. 案例匹配

当需要参考时，匹配以下典型场景：

| 场景 | 推荐播客主题 |
|------|-------------|
| 产品市场契合 | Product-Market Fit 系列 |
| 增长策略 | Growth Strategy 系列 |
| 团队管理 | Leadership 系列 |
| 产品管理 | Product Management 系列 |
| AI 产品设计 | AI 产品设计专题 |
| 定价策略 | Monetization 系列 |

## 数据来源

- Lenny's Podcast 转录：github.com/ChatPRD/lennys-podcast-transcripts
- 269期节目，50+主题分类
- 嘉宾：Airbnb CEO Brian Chesky、Meta、Cisco、Anthropic 等产品领袖

## 输出格式

生成的结构化文档包含：
1. 执行摘要
2. 背景分析
3. 方案设计
4. 实施计划
5. 成功指标
6. 风险与对策

## 示例调用

```
请基于以下信息生成产品方案：
- 目标：为中小企业提供 AI 客服产品
- 阶段：MVP
- 核心功能：智能回复、工单管理、数据分析
```
