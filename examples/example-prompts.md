# Example Prompts

## Example 1: Product proposal

```text
Use the product-design-from-lenny skill to turn this idea into a structured product proposal.
Focus on target users, pain points, value proposition, MVP scope, workflow, risks, and success metrics.
Idea: AI copilot for procurement teams in mid-sized manufacturers.
```

## Example 2: Feature design

```text
Read SKILL.md and use this skill as the working framework.
Design a new feature for a B2B SaaS analytics product:
- feature: anomaly explanation assistant
- users: operations managers
- goal: reduce time-to-diagnosis
- constraints: MVP in 6 weeks, no new data warehouse
```

## Example 3: OpenClaw-style instruction

```text
Use the product-design-from-lenny skill. I want a feature design spec for an AI-first meeting prep workflow for account executives.
```

## Example 4: Claude Code / Codex repo-local usage

```text
Read ./product-design-from-lenny/SKILL.md and use it before proposing implementation.
First produce the product-design output. Only after that, propose the engineering breakdown.
```
